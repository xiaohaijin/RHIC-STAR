The TpcResponseSimulator Online Information Page

This is a public www page to provide information about the TpcResponseSimulator. This package has its own home page.

For all STAR documentation, Go here

In order to obtain always the most up-to-date version do the following:

    mkdir wrk
    cd wrk
    cvs co StRoot/StTrsMaker
    cd StRoot/StTrsMaker/doc/tex
    make

This creates a PS file trs.ps which you can print easily. This requires that you have a complete version of LaTeX installed on your system.

Optionally you can also create directly a pdf file trs.pdf:

make pdf

This requires that you have the dvipdf program installed on you system.
