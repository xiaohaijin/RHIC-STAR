StStrangeMuDstMaker Documentation
Offline Software using ROOT in STAR Maintained by G. Van Buren
Last modified Tue Apr 11 15:30:19 2000

The StStrangeMuDstMaker package implements a micro-DST for the STAR Strangeness Physics Working Group using TTrees. Several example macros are provided in the STAR macros area and are described below, in addition to the operation and usage of the package itself.
  • The Micro-DST:
    The StrangeMuDst micro-DST is a TTree-based DST which stores components relating to different vertex types for strangeness studies.

  • The Maker:
    The StStrangeMuDstMaker maker runs in a standard STAR chain, and allows several modes for creating/reading/filtering strangeness micro-DSTs described herein.

  • The Formulas Macro:
    The strangeFormulas.C macro allows the user access to a complex, but efficient function set for use with the micro-DST.

  • The Cuts:
    Cuts are stored in the StrangeCuts branch (formerly stored as a TOrdCollection along with the micro-DST tree in the files) which enumerates the cuts used to produce the micro-DST.

  • The Player:
    The StStrangeMuDstPlayer is a new front-end to StrangeMuDstMaker, which provides a simplified user interface for making and filtering a micro-DST.

  • DcaService:
    DcaService is a tool for recalculating DCAs of various particles to the primary vertex in the micro-DST.
