  • The printlists of St_QA_Maker & StEventQAMaker

Macro Options (e.g., bfcread_hist_to_ps.C)

  • MakerHistDir
    • QA
          table DST-based QA histograms
    • EventQA
          StEvent-based QA histograms

  • PrintList
    • (no option, i.e., PrintList="")
          if no option is specified, then all QA histograms will be printed by default
    • qa_shift
          year 2000 run QA shift histograms set
    • year1
          all histograms except FTPC and SVT
    • Cosmic
          small subset of relevant QA histograms for cosmic data
    • Ftpc
          FTPC-only histograms
    • MDC3
          various flow histograms examined during MDC3
    • TestQATable
          example of a printlist

The darkblue printlist options are only available with the MakerHistDir="QA" option at the moment.
