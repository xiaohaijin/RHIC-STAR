FTPC Cluster Finder Maker

StFtpcClusterMaker

StFtpcClusterMaker replaces St_fcl_Maker in the FTPC production chain. It includes the functionality of both fcl and ffs, so StRoot/St_fcl_Maker, pams/ftpc/fcl and pams/ftpc/ffs (one of FTPCs last fortran modules) become obsolete. The change became necessary to prepare the transition from a table based raw data format to raw data reader classes.

If raw data (fcl_ftpcndx, fcl_ftpcsqndx and fcl_ftpcadc) is found, StFtpcClusterMaker loads it as input tables and the parameter tables (fcl_ampslope, fcl_ampoff, fcl_timeoff, fcl_padtrans, fcl_det and fcl_zrow) and initializes the output table (fcl_fppoint) for StFtpcClusterFinder .

If no raw data (from the daq or the slow simulator) is found, StFtpcClusterMaker then loads g2t_vertex, g2t_track and g2t_ftp_hit as input tables and the parameter tables (ffs_fspar and ffs_gaspar) and initializes the output tables (fcl_fppoint and ffs_gepoint) for the FTPC fast simulator StFtpcFastSimu.
